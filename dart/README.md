# Dart FFI: binding to dart.


- use `dart:ffi` to binding `wallet-core` for dart.


## quickstart:


- install:

```
make osx-install

```

- setup:

```
make ffi-setup

```

- gen binding codes:

```
make ffi-gen
```

## dart binding file:

- result: [dart/wallet_core_bindings.dart](./dart/wallet_core_bindings.dart)


## pubspec.yaml:
